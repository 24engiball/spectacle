Spectacle

Move and resize your windows with ease.

  • Center / Alt + Cmd + C
  • Fullscreen / Alt + Cmd + F
  • Left Half / Alt + Cmd + ←
  • Right Half / Alt + Cmd + →
  • Top Half / Alt + Cmd + ↑
  • Bottom Half / Alt + Cmd + ↓
  • Upper Left Corner / Ctrl + Cmd + ←
  • Lower Left Corner / Ctrl + Shift + Cmd + ←
  • Upper Right Corner / Ctrl + Cmd + →
  • Lower Right Corner / Ctrl + Shift + Cmd + →
  • Next Display / Ctrl + Alt + Cmd + ←
  • Previous Display / Ctrl + Alt + Cmd + →
  • Next Third / Ctrl + Alt + ←
  • Previous Third / Ctrl + Alt + →
  • Undo / Alt + Cmd + Z
  • Redo / Alt + Shift + Cmd + Z

About

Those familiar with similar apps will find Spectacle to be a simple and unobtrusive way to become more productive while using your Mac. Using Spectacle you can view multiple documents side-by-side, move windows to other displays, or even focus your full attention to a single task.

Spectacle has been, and will continue to be, an open source project. The source code can be found on GitHub.

Latest

Spectacle now requires OS X 10.7 or better. Please feel free to download and install Spectacle 0.7 if you are unable to use the latest release.
