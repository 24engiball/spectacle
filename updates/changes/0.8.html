Spectacle 0.8

This version of Spectacle provides the following fixes and enhancements:

  • Do not resize or move windows that already fill the dimensions of the current screen (issue #57)
  • Make it possible to blacklist applications that don't work well with Spectacle (issue #77)
  • Add "Adobe Reader" to the application blacklist (issue #77)
  • Moving a window backwards between thirds of the display does not always work (issue #93)
  • Increase and/or decrease the size of arbitrary windows (issue #42)
  • Windows should "stick" to the edges of the screen they touch when resizing (issue #92)

Important Note: Spectacle 0.8 now requires OS X 10.7 or better. Those still using OS X 10.6 should skip this update and continue to use the previous Spectacle release.
